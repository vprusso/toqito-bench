# toqito-bench
Benchmarking suite for the toqito software package.

## Setup Environments

### Python Environment
 - to setup a `python 3.10` virtual environment
```bash
make setup-python
```
- to run tests
```bash
make test-python
```
- to clean up
```bash
make clean-python
```

### Julia Environment
 - to setup `Julia 1.10` environment
```bash
make setup-julia
```
- to run tests
```bash
make test-julia
```
- to clean up
```bash
make clean-julia
```
